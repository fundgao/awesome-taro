## ⏬ Install

- [Taro](https://taro-docs.jd.com/docs/)

```bash
npm install -g @tarojs/cli

npm install

npm run dev:weapp

npm run build:weapp
```

## ⏬ Install Taro UI

- [Taro UI](https://taro-ui.jd.com/#/docs/quickstart)

```bash
npm install taro-ui
```

## 卸载 Taro
```
cd /usr/local/lib/node_modules
rm -rf \@tarojs/

安装指定版本
npm install -g @tarojs/cli@3.2.1

安装最新版
npm install -g @tarojs/cli@latest
```
