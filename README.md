# weapp-taro

taro for learn

# 📖 Introduction

- Node.js v18.13.0
- Taro v3.6.18

## ⏬ Install

- [Taro](https://taro-docs.jd.com/docs/)

```bash
npm install -g @tarojs/cli

npm install

npm run dev:weapp

npm run build:weapp
```

## ⏬ Install Taro UI

- [Taro UI](https://taro-ui.jd.com/#/docs/quickstart)

```bash
npm install taro-ui
```
